## Research Memo

**Questions Presented**

List the issue(s) in concise question form.

**Brief Answer**

State the answer in 1-3 paragraphs. Include the bottom-line risk level if the answer is uncertain.

**Governing Law**

State the rule statements. Every authority group should support a rule statement. Organize controlling rules/statutes/cases first and persuasive authority last. Avoid citation dumps.

**Analysis**

Explain how the authorities apply to the user's facts or draft issue.

**Risks / Objective Assessment**

Identify contrary authority, weak pincites, negative treatment, procedural uncertainty, or unresolved factual assumptions.

**Recommendation**

Provide concise language the user can use in the work product.

**Authorities Reviewed**

Use a pure citation list unless the user asks for treatment notes or an authority bank.

Do not include a "Lexis verification status" section in the attorney memo by default. Put verification status, search logs, timeout notes, export limitations, and rejected authorities in working files unless the user requests them in the memo.
